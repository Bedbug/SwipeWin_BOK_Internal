5. How to enter

5.1 Employees must log into the “Bok Trivia” platform using their oracle/staff number either on their phone or web via competition’s link.

5.2 Each round of the competition comprises of 15 questions and the users have 5 minutes to answer them all.

5.3 When the 5 minutes is over the round is over ,users receive points upon answering questions correctly .

5.4 Points are awarded as follows:

5.4.1 Twenty (20) points if this is a “common” type question

5.4.2 Fifty (50) points if this is a “bronze” type question

5.4.3 One hundred points (100) if this is a “silver” type question

5.4.4 One hundred and fifty points (150) if this is a “golden” type question

5.5 Bronze, silver and gold questions award extra points and pop up at surprise intervals
